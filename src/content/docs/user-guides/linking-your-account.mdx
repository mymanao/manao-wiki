---
title: การเชื่อมบัญชี
---

สิ่งที่บอตมะนาวโดดเด่นกว่าบอตอื่น ๆ (แหละมั้ง) คือสามารถเชื่อมข้อมูลบัญชีแบบหลายแพลตฟอร์มได้ โดยตอนนี้รองรับการเชื่อม Twitch, Kick และ Discord เข้าหากัน

คู่มือนี้จะมาสอนวิธีเชื่อมบัญชีของเรากับแพลตฟอร์มต่าง ๆ ผ่านคำสั่ง `link` นั่นเอง

### วิธีการเชื่อมบัญชี

ระบบการเชื่อมบัญชีของมะนาวทำงานแบบ **รหัสคู่** คือแพลตฟอร์มนึงสร้างรหัส แล้วอีกแพลตฟอร์มนึงนำรหัสนั้นมาใช้เพื่อเชื่อมบัญชีเข้าหากัน โดยไม่จำเป็นต้องเริ่มจาก Discord เสมอไปแล้ว!

### สร้างรหัสเชื่อมต่อ

สามารถสร้างรหัสได้จาก **ทุกแพลตฟอร์ม** ที่มะนาวรองรับ:

**Twitch** — พิมพ์ในแชท:
```
!link
```

**Kick** — พิมพ์ในแชท:
```
!link
```

**Discord** — พิมพ์ slash command:
```
/link
```

บอตจะตอบกลับมาด้วยรหัส 6 ตัวอักษร เช่น `X7K2PQ`

:::tip
รหัสนี้จะมีอายุ **1 นาที** เท่านั้น ถ้าเลยเวลาต้องสร้างใหม่
และใช้ได้ครั้งเดียวเท่านั้น หลังจากใช้แล้วจะถูกลบทันที
:::

### เชื่อมบัญชี

เมื่อได้รหัสมาแล้ว ให้นำไปใช้ในอีกแพลตฟอร์มที่ต้องการเชื่อม:

**Twitch / Kick** — พิมพ์ในแชท:
```
!link [รหัส]
```

**Discord** — พิมพ์ slash command:
```
/link [รหัส]
```

แค่นี้ก็เสร็จแล้วจ้า!

### ตัวอย่างการใช้งาน

**เชื่อม Twitch กับ Discord:**
1. พิมพ์ `!link` ในแชท Twitch → ได้รหัส `X7K2PQ`
2. พิมพ์ `/link X7K2PQ` ใน Discord → เชื่อมสำเร็จ ✅

**เชื่อม Kick กับ Discord:**
1. พิมพ์ `!link` ในแชท Kick → ได้รหัส `A3BF9Z`
2. พิมพ์ `/link A3BF9Z` ใน Discord → เชื่อมสำเร็จ ✅

**เชื่อม Twitch กับ Kick:**
1. พิมพ์ `!link` ในแชท Twitch → ได้รหัส `M5QR2W`
2. พิมพ์ `!link M5QR2W` ในแชท Kick → เชื่อมสำเร็จ ✅

:::note
บัญชีที่เชื่อมกันแล้วจะแชร์ข้อมูลร่วมกัน เช่น ยอดเงิน และชื่อเล่น
:::
