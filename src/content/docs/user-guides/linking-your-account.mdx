---
title: การเชื่อมบัญชี
---

มะนาวสามารถเชื่อมบัญชีข้ามแพลตฟอร์มได้ รองรับ Twitch, Kick, Discord และ YouTube เมื่อเชื่อมแล้ว ยอดเงินและชื่อเล่นจะแชร์ร่วมกันทุกแพลตฟอร์ม

ระบบทำงานแบบ **รหัสคู่** คือแพลตฟอร์มหนึ่งสร้างรหัส แล้วอีกแพลตฟอร์มนำรหัสนั้นมาใช้เพื่อเชื่อม

---

### สร้างรหัสเชื่อมต่อ

สร้างรหัสได้จาก **ทุกแพลตฟอร์ม** ที่มะนาวรองรับ:

**Twitch / Kick / YouTube** — พิมพ์ในแชท:
```
!link
```

**Discord** — พิมพ์ slash command:
```
/link
```

บอตจะตอบกลับด้วยรหัส 6 ตัวอักษร เช่น `X7K2PQ`

:::tip
รหัสนี้จะมีอายุ **1 นาที** เท่านั้น ถ้าเลยเวลาต้องสร้างใหม่ และใช้ได้ครั้งเดียวเท่านั้น
:::

---

### เชื่อมบัญชี

นำรหัสไปใช้ในอีกแพลตฟอร์มที่ต้องการเชื่อม:

**Twitch / Kick / YouTube** — พิมพ์ในแชท:
```
!link [รหัส]
```

**Discord** — พิมพ์ slash command:
```
/link [รหัส]
```

---

### ตัวอย่างการใช้งาน

**เชื่อม Twitch กับ Discord:**
1. พิมพ์ `!link` ในแชท Twitch → ได้รหัส `X7K2PQ`
2. พิมพ์ `/link X7K2PQ` ใน Discord → เชื่อมสำเร็จ ✅

**เชื่อม Kick กับ YouTube:**
1. พิมพ์ `!link` ในแชท Kick → ได้รหัส `A3BF9Z`
2. พิมพ์ `!link A3BF9Z` ในแชท YouTube → เชื่อมสำเร็จ ✅

**เชื่อม Twitch กับ Kick:**
1. พิมพ์ `!link` ในแชท Twitch → ได้รหัส `M5QR2W`
2. พิมพ์ `!link M5QR2W` ในแชท Kick → เชื่อมสำเร็จ ✅

:::note
บัญชีที่เชื่อมกันแล้วจะแชร์ยอดเงินและชื่อเล่นร่วมกัน
:::
