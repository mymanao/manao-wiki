title: เริ่มใช้งาน Dashboard
---

เมื่อรัน ManaoBot แล้ว จะมีหน้าต่าง Dashboard เปิดขึ้นมาใน Browser โดยสามารถเข้าถึงได้ที่ `http://localhost:3000` หน้านี้จะเป็นศูนย์กลางในการจัดการบอตของเรา สามารถดูสถานะต่าง ๆ และปรับแต่งการตั้งค่าได้จากที่นี่

:::tip[เปิดในเครื่องอื่น]
ถ้าอยากเปิด Dashboard ในเครื่องอื่นที่เชื่อมต่อกับเครือข่ายเดียวกัน ให้ใช้ IP Address ของเครื่องที่รัน Manao แทน `localhost`

ซึ่งตอนรัน Manao จะแสดง IP อยู่แล้ว เช่น `http://192.168.1.199:4600` ให้ใช้ลิงก์นี้เปิดในเครื่องอื่นได้เลย (ต้องอยู่ใน Wi-Fi เดียวกันนะ)
:::

### Commands

ในหน้านี้สามารถเปิด/ปิดคำสั่งต่าง ๆ ได้ นอกจากนั้นยังสร้างคำสั่งเองได้ด้วย ดูรายละเอียดได้ที่ [เขียนคำสั่งแรก](/guides/writing-your-first-custom-command)

---

### Soundboard

ในหน้านี้สามารถเปิดเสียงเอฟเฟกต์ต่าง ๆ ได้ เสียงจะถูกเล่นในหน้า Sound Overlay ซึ่งสามารถเพิ่มเข้าไปใน OBS ได้ง่าย ๆ ดูวิธีการเพิ่มโอเวอร์เลย์ใน OBS ได้ที่ [เพิ่มโอเวอร์เลย์ใน OBS](/guides/adding-overlays-in-obs)

**การเพิ่มเสียงใหม่**

1. วางไฟล์เสียงไว้ในโฟลเดอร์ `server/public/sounds/` ในโฟลเดอร์มะนาว
2. ไปที่หน้า Soundboard แล้วกด **Add Sound**
3. ตั้งชื่อ และใส่ URL เป็น `/sounds/[ชื่อไฟล์].[นามสกุล]` เช่น `/sounds/vine-boom.mp3`

หรือใส่เ่ป็นลิงก์เสียงจากภายนอกก็ได้ เช่น `https://www.myinstants.com/media/sounds/taco-bell.mp3`

### Config

ปรับการตั้งค่าต่าง ๆ ของบอตได้โดยไม่ต้องแก้ไฟล์เอง

**ทั่วไป**
- **Language**: ภาษาที่บอตจะใช้คุย (`en` / `th`)
- **Currency Name**: ชื่อหรืออีโมจิที่ใช้เป็นสกุลเงิน (เช่น `COIN`, `🪙`)

**Command Prefixes**
- ตั้งค่า prefix แยกกันได้ในแต่ละแพลตฟอร์ม (Twitch, Kick, Discord, YouTube) ค่าเริ่มต้นคือ `!`

**Chat Rewards**

ตั้งค่าเงินที่ผู้ชมจะได้รับเพียงแค่พิมพ์แชท แยกตั้งค่าได้ในแต่ละแพลตฟอร์ม:

- **Min / Max**: ช่วงเงินที่คนดูจะได้ (เช่น 1-4)
- **Chance**: เปอร์เซ็นต์โอกาสที่จะได้รับเงิน (เช่น 50 หมายความว่ามีโอกาส 50% ที่จะได้รับเงินในช่วง 1-4 เมื่อพิมพ์แชท)
- **Cooldown (s)**: เวลาที่ต้องรอก่อนที่จะได้รับเงินอีกครั้ง (เช่น 60 หมายความว่าถ้าคนดูได้รับเงินแล้ว จะต้องรอ 60 วินาทีถึงจะมีโอกาสได้รับเงินอีกครั้ง)

**Custom Messages**

ปรับข้อความที่มะนาวส่งเมื่อมีเหตุการณ์ รองรับ EN/TH: `onFollow`, `onSubscribe`, `onGiftSubscribe`, `onRaid`, `onCheer`
ใช้ `[user]` แทนชื่อผู้ชม และ `[viewers]` แทนจำนวนคนที่เข้ามาบุก

- **onFollow**: ข้อความที่ส่งเมื่อมีคนติดตามใหม่
- **onSubscribe**: ข้อความที่ส่งเมื่อมีคนซับใหม่
- **onGiftSubscribe**: ข้อความที่ส่งเมื่อมีคนกิ๊ฟต์ซับ
- **onRaid**: ข้อความที่ส่งเมื่อมีคนบุกช่อง
- **onCheer**: ข้อความที่ส่งเมื่อมีคนบิต

### Managers

หมวดนี้มีหน้าย่อยสำหรับจัดการ:
- **Custom Commands**: สร้าง/แก้ไข/ลบคำสั่งที่สร้างเอง
- **Custom Replies**: ตั้งค่าข้อความตอบกลับอัตโนมัติ เมื่อมีคนพิมพ์คำที่กำหนดในแชท
- **Channel Points**: เชื่อมแต้มช่อง Twitch กับ Soundboard (แลกแต้มเพื่อเล่นเสียง)
- **Scheduled Messages**: ตั้งรายการของข้อความที่ส่งอัตโนมัติตามเวลา (เช่น ทุก ๆ 3 นาทีให้ส่งข้อความโปรโมทโซเชียลมีเดีย)
