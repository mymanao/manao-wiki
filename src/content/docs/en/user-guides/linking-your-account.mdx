---
title: Linking Your Account
---

Manao can link accounts across platforms — Twitch, Kick, Discord, and YouTube. Once linked, your balance and nickname are shared across all platforms.

Linking uses a **code-pair system**: one platform generates a code, and the other uses it to link.

---

### Generate a Link Code

From **any supported platform**:

**Twitch / Kick / YouTube** — type in chat:
```
!link
```

**Discord** — use slash command:
```
/link
```

The bot replies with a 6-character code like `X7K2PQ`.

:::tip
Codes expire after **1 minute** and can only be used once.
:::

---

### Link Your Account

Use the code on the other platform:

**Twitch / Kick / YouTube** — type in chat:
```
!link [code]
```

**Discord** — slash command:
```
/link [code]
```

---

### Examples

**Link Twitch ↔ Discord:**
1. Type `!link` in Twitch chat → get code `X7K2PQ`
2. Type `/link X7K2PQ` in Discord → linked ✅

**Link Kick ↔ YouTube:**
1. Type `!link` in Kick chat → get code `A3BF9Z`
2. Type `!link A3BF9Z` in YouTube chat → linked ✅

**Link Twitch ↔ Kick:**
1. Type `!link` in Twitch chat → get code `M5QR2W`
2. Type `!link M5QR2W` in Kick chat → linked ✅

:::note
Linked accounts share balance and nickname across all platforms.
:::
