## ทำไมต้องมะนาว?

<CardGrid stagger>
  <Card title="คำสั่งเพลง" icon="star">
    ให้ผู้ชมของคุณขอเพลงได้โดยตรงจากแชทผ่านคำสั่งต่าง ๆ มาพร้อมกับโอเวอร์เลย์เพลงในตัว รองรับทั้ง Twitch, Kick และ YouTube!
  </Card>

  <Card title="โอเวอร์เลย์" icon="comment">
    มีโอเวอร์เลย์แชทและโอเวอร์เลย์ต่าง ๆ พร้อม theme ให้เลือกหลายแบบ และปรับแต่งได้ตามใจผ่านหน้า Overlay Settings
  </Card>

  <Card title="ซาวด์บอร์ด" icon="rss">
    เรียกใช้เสียงเอฟเฟกต์ได้ทันทีผ่านซาวด์บอร์ด หรือให้คนดูแลกผ่านแต้มช่องบน Twitch
  </Card>

  <Card title="เพิ่มคำสั่งใหม่" icon="setting">
    เพิ่มคำสั่งที่สร้างขึ้นเองได้ผ่าน dashboard เขียน JavaScript ได้ตามต้องการ ใช้งานได้ทุกแพลตฟอร์มในคราวเดียว
  </Card>

  <Card title="โอเพ่นซอร์ส" icon="open-book">
    มะนาวบอตเป็นโปรแกรมโอเพ่นซอร์สและฟรี ปลอดภัย ยืดหยุ่น ปรับแต่งเองได้เยอะมาก
  </Card>

  <Card title="รองรับหลายแพลตฟอร์ม" icon="discord">
    รองรับ Twitch, Kick, Discord และ YouTube ในคราวเดียว และสามารถเชื่อมบัญชีข้ามแพลตฟอร์มได้ด้วย
  </Card>
</CardGrid>
